Creates a Spinner to use for operations that run for some time and can be used to give the user the indication that the process is taking some time, but still running. Most of the work in this class is performed by the progress module (found on PyPi, written by Giorgos Verigakis) This is just a thin wrapper to used by this framework so to reduce repitition and provide some customizations.
